# Nilüfer Konak Cafe Reservation System ☕️

This is a web project I developed for our family-run cafe in Bursa to transition their manual reservation process into a digital platform. 

## 📖 About the Project
The goal was to create a lightweight, fast, and user-friendly interface for customers to book tables online. I focused on accessibility so that customers of all ages could easily navigate the site and reach out to the business.

## ✨ Key Features
- **Live Booking:** Connected to a **Supabase** backend to store and manage reservations in real-time.
- **Mobile First:** Fully responsive design that works perfectly on smartphones.
- **Direct Contact:** Integrated WhatsApp, Instagram, and professional email links.
- **Dynamic Maps:** Built-in Google Maps for easy location tracking.

## 🛠 Tech Stack
- **Frontend:** Plain HTML5, CSS3 (Flexbox & Grid), and Vanilla JavaScript.
- **Backend:** Supabase (Database & API management).
- **Hosting:** Deployed via GitHub Pages.

## 🔒 Note on Security
The Supabase URL and API keys have been removed from the public repository to prevent unauthorized access. In a production environment, these are handled as environment variables.
